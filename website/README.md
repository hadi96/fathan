# Website Laporan Pertukaran Pelajar

Template sederhana untuk laporan pertukaran pelajar dengan tema ungu yang bersih dan UI/UX responsif.

## Fitur

- Tema warna ungu (#6a0dad) yang konsisten di seluruh halaman
- Navigasi sederhana dengan anchor link ke bagian
- Bagian untuk deskripsi program, kegiatan, galeri foto, dan kontak
- Grid responsif untuk menampilkan gambar
- Mudah dikustomisasi dengan mengganti teks dan menambahkan foto

## Cara Menggunakan

1. Salin/unggah foto kegiatan ke folder `assets/images/` dan ganti nama placeholder.
2. Edit teks pada `index.html` sesuai dengan laporan kamu.
3. Buka `index.html` di peramban untuk melihat pratinjau.
4. Tambahkan CSS atau JavaScript tambahan jika diperlukan.
